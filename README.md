# Advent of Code

Personal solutions for [Advent of Code](https://adventofcode.com/) in [Zig](https://ziglang.org/).

## How to run

`zig build run -- <year> <day> <a|b>`

`zig build test -- <year> <day> <a|b>`

`zig test <path_to_file.zig>`
